Step by step create “dao service controller”

  • step1. Model

    • create model folder
    • create “User” class in model folder
    • example

  • step2. Dao

    • create “dao” folder
    • create “UserDao” interface in “dao” folder
    • create “impl” folder in “dao” folder
    • create “UserDaoImpl” class in “impl” folder

  • step3. Service

    • create “service” folder
    • create “UserService” interface in “service” folder
    • create “impl” folder in “service” folder
    • create “UserServiceImpl” class in “impl” folder

  • step. Controller

    • create “controller” folder
    • create “UserController” class in “controller” folder
